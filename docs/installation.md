# Installation

## Stable release

To install inapps-pybackend, run this command in your terminal:

```sh
uv add inapps-pybackend
```

Or if you prefer to use `pip`:

```sh
pip install inapps-pybackend
```

## From source

The source files for inapps-pybackend can be downloaded from the [Github repo](https://github.com/jawad-khan/inapps_pybackend).

You can either clone the public repository:

```sh
git clone git://github.com/jawad-khan/inapps_pybackend
```

Or download the [tarball](https://github.com/jawad-khan/inapps_pybackend/tarball/master):

```sh
curl -OJL https://github.com/jawad-khan/inapps_pybackend/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd inapps_pybackend
uv pip install .
```
